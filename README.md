# Movix 
Movix is clone of hotStar.com . It is a most UI based  website using HTML SCSS & Javascript & React

Team members: 
1. Shaguin Sharma

TechStack:
1. React
2. React-Router-Dom
3. Redux-ToolKit
4. Scss
5. Day JS (npm package for formatiing date)
6. react-circular-progressbar (for rating progress)
